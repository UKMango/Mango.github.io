Python--初学者教程【2】

2022-06-12

《1》安装Python

【1】Windows与MacOS用户安装Python3教程.

【注意！】下面有Linux用户的安装方法

1.Python下载链接:https://www.python.org/downloads/ 光标停靠在Downloads也就是下载这个选项上.

2.然后你能看见熟悉的操作系统，Windows，还有MacOS，选择与你系统相符合的.

3.点击进去，这里以Windows为例，会出现两个选项，这里看你系统是64还是32位的.

4.选择后，就下载了，我们打开安装程序EXE，记得勾选Add Python 3.9 to PATH，以后在Path 环境变量中，会很方便，然后在安装页面选择第一个，Install也就是安装，然后耐心等待安装结束.

5.安装结束后，用营销号说的😂Win+R，然后输入Cmd，调出命令符，然后输入Python，有提示就是成功了！😋

6.你以为这样就完美了?不不不😧还需要验证下，在Python控制台里输入打印Hello word或者打印一句你喜欢的句子.

7.到此Windows安装Python完美结束了！🎉😊

【2】Linux用户安装Python3教程.

【注意！】Linux安装还有另一种方法，在文章的后面

1.Linux用户安装方法:有些会自带Python，有些没有，没有的在终端里安装下依赖环境.

安装命令:yum -y install zlib-devel bzip2-devel openssl-devel ncurses-devel
sqlite-devel readline-devel tk-devel gdbm-devel db4-devel libpcap-devel
xz-devel

2.然后下载Python.

安装Wfet有安装的不用再重装:yum -y install wget

然后安装Python3:wget https://www.python.org/ftp/python/3.7.5/Python-3.7.5.tgz

3.解压.

tar -xzvf Python-3.7.5.tgz

4.安装.

安装Gcc:yum install gcc -y
3.7版本之后需要一个新的包:yum install libffi-devel -y
打开解压路径:cd Python-3.7.5
配置指定安装目录:./configure --prefix=/home/python
编译和编译安装:make && make install

5.配置Python环境变量.
export PYTHON_HOME=/home/python

export PATH=PATH:{PATH}:PATH:{PYTHON_HOME}/bin

在/etc/profile文件中加入上面两行内容:vi /etc/profile
重新加载配置:source /etc/profile

6.安装python虚拟环境.

新建名为python_venv的虚拟环境目录:python3 -m venv python_venv
进入虚拟环境所在文件夹的bin目录:cd python_venv/bin
激活虚拟环境:source activate

7.配置pip国内安装镜像源.

pip install -i https://pypi.tuna.tsinghua.edu.cn/simple(镜像地址)

8.其他镜像源:

8.批量安装Python项目依赖包.

导出当前环境:pip freeze > requirements.txt
下载依赖包:pip download -r requirements.txt -d

9.通过shell脚本一键安装.

#!/bin/sh
yum -y install wget
yum -y install gcc
yum -y install zlib-devel bzip2-devel openssl-devel ncurses-devel sqlite-devel readline-devel tk-devel gdb
m-devel db4-devel libpcap-devel xz-devel
yum -y install libffi-devel
if [ ! -f "Python-3.7.5.tgz" ]
then
echo Python-3.7.5.tgz not exist!!!
wget https://www.python.org/ftp/python/3.7.5/Python-3.7.5.tgz
fi `tar -zxvf Python-3.7.5.tgz`
if [ ! -d /home/python3 ]
then `mkdir -p /home/python3`
cd Python-3.7.5
./configure --prefix=/home/python3
make && make install
fi
echo "export PYTHON_HOME=/home/python3" >>/etc/profile
echo "export PATH=\${PATH}:\${PYTHON_HOME}\"/bin\"" >>/etc/profile
echo "=====`date`===== Python-3.7.5 install success !!!====="

10.😃到此Linux安装Pyhton3即安装成功了！

Linux用户安装Pyhton另一种方法.

1.有些Linux系统可以有些不行，不行的用上面那种方法. 2.Python3.8

3.Python2.7

终端输入sudo apt-get install python2.7

4.🤗同样也可以安装成功！
